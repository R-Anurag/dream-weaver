# **App Name**: Dream Weaver

## Core Features:

- Flexible Canvas: Provide a flexible digital canvas where users can freely arrange and manipulate visual elements.
- Image Import and Manipulation: Enable users to add images from local storage, resize them, and position them freely on the canvas.
- Shape Insertion and Customization: Offer a selection of customizable shapes (rectangles, circles, stars, etc.) that users can insert and style.
- Text Box Creation and Editing: Allow users to create and format text boxes with adjustable fonts, sizes, and colors.
- Digital Post-it Notes: Include digital post-it notes that users can place on the board for reminders and affirmations.
- Board Saving: Implement a saving mechanism so users can store their boards locally.
- Board Organization: Allow users to organize multiple vision boards within the app.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to create a sense of calm and serenity, relating to themes of relaxation, contemplation, and goal-setting.
- Background color: Light pastel gray (#F0F0F0), almost white but not stark, similar hue to lavender but desaturated
- Accent color: Peach (#FFB380) to create a focal point, adding an element of cheerfulness, with a higher contrast to the desaturated background than the primary color.
- Body and headline font: 'Alegreya', a serif with an elegant, intellectual, contemporary feel.
- Use simple, outlined icons for a clean and modern look.
- Maintain a clean and open layout to prevent clutter and enhance focus on the content.
- Incorporate subtle animations for interactive elements to provide gentle feedback and engagement.
