# Dream Weaver

Dream Weaver is a full-stack, responsive web application that provides a creative digital canvas for users to build and share their vision boards. It's designed to help you visualize your goals, manifest your dreams, and connect with a community of creators.

From personal mood boards to collaborative project planning, Dream Weaver uses a modern tech stack to deliver an intuitive and engaging user experience, complete with AI-powered tools to spark inspiration.

## Key Features

*   **Intuitive Canvas:** A drag-and-drop interface to add and manipulate images, text, sticky notes, and shapes.
*   **AI-Powered Proposals:** Leverage Google's Gemini models via Genkit to generate creative collaboration proposals.
*   **Explore & Connect:** Discover public vision boards, find inspiration, and connect with potential collaborators.
*   **Responsive Design:** A seamless experience on both desktop and mobile devices.
*   **Component-Based UI:** Built with reusable and accessible components.
*   **Client-Side Persistence:** Boards are saved in `localStorage` for a fast, client-first experience.

## Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/) (App Router)
*   **Language:** [TypeScript](https://www.typescriptlang.org/)
*   **UI Components:** [ShadCN UI](https://ui.shadcn.com/)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Generative AI:** [Genkit (Google)](https://firebase.google.com/docs/genkit)
*   **Icons:** [Lucide React](https://lucide.dev/)
*   **Animations:** [Lottie](https://lottiefiles.com/), [Framer Motion](https://www.framer.com/motion/) (via `tailwindcss-animate`)
*   **Deployment:** Firebase App Hosting
